I am content included from an external file. I need webpack to load me. React doesn't like external HTML so much, so i am wrapped in a container with dangerouslySetInnerHTML, like this: 

import includedContent from './some/html/file.html'
<div
	dangerouslySetInnerHTML={{
		__html: includedContent
	}}
/>
This example uses html-loader plugin for webpack.
